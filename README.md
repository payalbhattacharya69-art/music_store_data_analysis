🎵 Music Store Data Analysis using SQL
📌 Project Overview

This project analyzes a music store database using pgAdmin 4 and PostgreSQL.
It focuses on solving real-world business problems using SQL queries ranging from basic to advanced level.

🎯 Objectives
Perform data analysis using SQL
Solve business-related queries
Understand customer behavior and sales trends
Practice joins, aggregations, and subqueries

🛠️ Tools & Technologies
Database: PostgreSQL
Tool Used: pgAdmin 4
Language: SQL

📂 Dataset Description
The dataset contains information about:
Customers
Invoices & Invoice Items
Tracks, Albums, Artists
Genres
📊 Analysis & SQL Queries

🔹 Basic Level Queries
Senior most employee based on job title
Countries with the most invoices
Top 3 invoice totals
City generating highest revenue
Best customer (highest spender)

🔸 Moderate Level Queries
Rock music listeners (email, name, genre)
Artists with most rock music tracks
Tracks longer than average song length

🔺 Advanced Level Queries
Amount spent by each customer on artists
Most popular genre by country
Top customer by country (highest spending)
